Introduo:

Este relatrio foi elaborado no contexto da unidade curricular de Fundamentos de Programao. E visa fornecer uma anlise estatstica informal ao Comit dos Jogos Olmpicos relativa aos eventos e atletas que competiram.
A anlise foi feita relativamente modalidade de Luge, no segmento de Corrida Individual Masculina. Esta anlise compreende os eventos ocorridos em 2002, 2006, 2010, 2014 e 2018; bem como a informao relativa a todos os atletas que participaram na modalidade.
O relatrio composto por uma breve descrio da modalidade, seguido da descrio da anlise realizada, descrio dos principais resultados e concluses da anlise e por fim as referncias.


Luge


Descrio:

Luge um desporto de Inverno o qual utiliza o tren para fazer uma descida numa pista de gelo. O piloto desce a pista deitado no tren, deitado de costas e com a cabea para trs. Esta forma permite ao piloto atingir uma grande velocidade na pista, contudo com uma viso reduzida da pista. A velocidade mxima do tren vaia entre 120 km.h a 155 km/h. A competio nesta modalidade feita individualmente ou em duplas. Carateristicas gerais da competio nos Jogos olimpicos:

    Pista:
  • Tamanho: 1500 metros
  • Nmero de curvas: 15 a 19
    Qualificao:
  • Realizadas quatro corridas entre 2 dias
  • Cada corrrida tem o seu ranking
  • O vencedor o que obtm o melhor tempo somando as quatro corridas

Ao observar os dados, percebemos que era possvel adicionar novos dados, como a velocidade da corrida, uma vez que tnhamos o tamanho da pista.
Para isso foi criada uma funo que calcula a velocidade (velocidade = distncia / tempo) de cada jogador em cada ronda, ou seja, por cada linha da dataframe, e adiciona o valor recm-calculado a uma lista.
Esta lista so todos os valores de velocidade.

In [116]:
def column_velocity(dataframe):
    velocity = []
    lista_tempos = dataframe['finish_time_seconds'].tolist()
    for i in range(len(lista_tempos)):
        # Pista = 1500 metros
        value = 1500/lista_tempos[i]
        velocity.append(value)
    return velocity

velocidades_lista = column_velocity(jogos_olimpicos)
jogos_olimpicos['velocity (m/s)'] = velocidades_lista

import unittest
from analise import column_velocity
import os
import pandas as pd


class MyTest(unittest.TestCase):
    
    def test_athlete_file_exist(self):
        print('\nTesting if athlete.csv file exists...')
        self.assertTrue(os.path.exists('athletes.csv'))
        print("PASS")
    
    def test_sport_events_file_exists(self):
        print('Testing if sport_events.csv file exists...')
        self.assertTrue(os.path.exists('sport_events.csv'))
        print("PASS")
    
    def test_with_1_rows(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5}])
        print('Testing column_velocity with 1 row...')
        self.assertEqual(column_velocity(dataframe),[300])
        print("PASS")

    def test_with_3_rows(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5},\
                                         {1: '1', 'd': 100, 'finish_time_seconds': 5},\
                                         {2: '2', 'd': 100, 'finish_time_seconds': 10}])
        print('Testing column_velocity with 3 rows...')
        self.assertEqual(column_velocity(dataframe),[300,300,150])
        print("PASS")

    
    def test_with_2_duplicate_columns_same_values(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5,'finish_time_seconds': 5},\
                                         {1: '1', 'd': 100, 'finish_time_seconds': 5, 'finish_time_seconds': 5},\
                                         {2: '2', 'd': 100, 'finish_time_seconds': 10, 'finish_time_seconds': 10}])
        print('Testing column_velocity result with duplicate columns with same values...')
        self.assertEqual(column_velocity(dataframe), [300,300,150])
        print("PASS")
        
    def test_with_2_duplicate_columns_different_values(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5,'finish_time_seconds': 5},\

                                         {2: '2', 'd': 100, 'finish_time_seconds': 10, 'finish_time_seconds': 10}])
        print('Testing column_velocity result with duplicate columns with different values...')
        self.assertEqual(column_velocity(dataframe), [300,150,150])
        print("PASS")
