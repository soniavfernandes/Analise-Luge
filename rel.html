Introdução:

Este relatório foi elaborado no contexto da unidade curricular de Fundamentos de Programação. E visa fornecer uma análise estatística informal ao Comité dos Jogos Olímpicos relativa aos eventos e atletas que competiram.
A análise foi feita relativamente à modalidade de Luge, no segmento de Corrida Individual Masculina. Esta análise compreende os eventos ocorridos em 2002, 2006, 2010, 2014 e 2018; bem como a informação relativa a todos os atletas que participaram na modalidade.

Luge


Descrição:

Luge é um desporto de Inverno o qual utiliza o trenó para fazer uma descida numa pista de gelo. O piloto desce a pista deitado no trenó, deitado de costas e com a cabeça para trás. Esta forma permite ao piloto atingir uma grande velocidade na pista, contudo com uma visão reduzida da pista. A velocidade máxima do trenó vaia entre 120 km.h a 155 km/h. A competição nesta modalidade é feita individualmente ou em duplas. Carateristicas gerais da competição nos Jogos olimpicos:

    Pista:
  • Tamanho: 1500 metros
  • Número de curvas: 15 a 19
    Qualificação:
  • Realizadas quatro corridas entre 2 dias
  • Cada corrrida tem o seu ranking
  • O vencedor é o que obtém o melhor tempo somando as quatro corridas

Cabeçalho: year,athlete_id,round_id,finish_time_seconds,rank_id

round_id String Número da corrida. round 1;round 2;round 3;round 4
finish_time_seconds Float Tempo em cada corrida [A:Z-a-z]
rank_id Integer Lugar do atleta [1:(Nr de atletas que participaram no respectivo ano)]

Desta forma, as colunas "year";"athlete_id";"round_id";"rank_id";"athlete_name";"date_of_birth";"country_sport_id" devem ser alteradas para dados categóricos. Dados categóricos são um tipo de variavel existente na biblioteca Pandas que permite uma ordenação dos elementos, ou seja, cada valor aquire valores que podem ser ordenados mas não permite calculos estatisticos entre os elementos.

Por exemplo, não faz sentido médias dos número de identificação dos atletas. O que faz sentido é uma ordenação, a criação de níveis.
Dessa forma é possível corrigir o tipo de dados com o método .astype() do pandas.

Manipulação dos dados

Com os dados do dataset é possível criar novos dados, agregar e analisar com outros "olhos" todo o conjunto de dados.

Ao observar os dados, percebemos que era possível adicionar novos dados, como a velocidade da corrida, uma vez que tínhamos o tamanho da pista.
Para isso foi criada uma função que calcula a velocidade (velocidade = distância / tempo) de cada jogador em cada ronda, ou seja, por cada linha da dataframe, e adiciona o valor recém-calculado a uma lista.
Esta lista são todos os valores de velocidade.

In [116]:
def column_velocity(dataframe):
    velocity = []
    lista_tempos = dataframe['finish_time_seconds'].tolist()
    for i in range(len(lista_tempos)):
        # Pista = 1500 metros
        value = 1500/lista_tempos[i]
        velocity.append(value)
    return velocity

velocidades_lista = column_velocity(jogos_olimpicos)
jogos_olimpicos['velocity (m/s)'] = velocidades_lista

Análise entre anos

Para isso, é aplicado às diferentes sub-dataframes da primeira corrida (utilizar apenas uma corrida facilita os calculos) o método .unique() (durante a iteração, este método sinaliza o elemento com True os que são únicos e False os não unicos), associado ao método .tolist() de forma a criar uma lista de verdadeiros, e a esta lista aplico a função len(), permitindo conhecer o númedo de valores unicos.

Resultados e Conclusões da Análise

Análise entre anos

Na contagem dos atletas, verificamos que houve uma participação maior no ano de 2002 e nos restantes anos, houve uma similaridade.

    Na análise das participações individuais dos atletas, verificamos que:
  • o atleta Shiva Keshavan participou em todas os eventos em estudo;
  • cerca de 5 atletas participaram em quatro dos eventos;
  • cerca de 20 participaram em três dos eventos;
    Na contagem de atletas que competem por determinado país verificamos que:
  • Austria, Canada, Alemanha, Italia, Romenia, Russia e Estados Unidos são os países que têm mais atletas a participar nos eventos em estudo;
  • A Ucrania nos últimos dois eventos, mostrou-se um país que tem levado mais atletas a participar

Análise entre corridas

    Na análise do tempo de corrida verica-se que intra anos, os valores são semelhantes, contudo, entre anos:
  • os valores de 2002 estão entre 45.882021 (max), 46.016224 (min)
  • os valores de 2006 estão entre 54.271500 (max), 52.382143 (min)
  • os valores de 2010 estão entre 49.544368 (max), 49.172000 (min)
  • os valores de 2014 estão entre 53.401077 (max), 52.677718 (min)
  • os valores de 2018 estão entre 48.360100 (max), 47.807700 (min)
    Na análise da velocidade verifica-se que intra anos, os valores são semelhantes, contudo, entre anos:
  • os valores de 2002 estão entre 32.748259 m/s (max), 32.503961 (min)
  • os valores de 2006 estão entre 28.642152 (max), 27.998112 (min)
  • os valores de 2010 estão entre 30.512362 (max), 30.283423 (min)

  • os valores de 2018 estão entre 31.376139 (max), 30.637459 (min)

No geral, é na corrida quatro (última) que maiores velocidades são atingidas, o que faz todo o sentido, visto ser a última tentativa para se qualificarem.

import unittest
from analise import column_velocity
import os
import pandas as pd


class MyTest(unittest.TestCase):
    
    def test_athlete_file_exist(self):
        print('\nTesting if athlete.csv file exists...')
        self.assertTrue(os.path.exists('athletes.csv'))
        print("PASS")
    
    def test_sport_events_file_exists(self):
        print('Testing if sport_events.csv file exists...')
        self.assertTrue(os.path.exists('sport_events.csv'))
        print("PASS")
    
    def test_with_1_rows(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5}])
        print('Testing column_velocity with 1 row...')
        self.assertEqual(column_velocity(dataframe),[300])
        print("PASS")

    def test_with_3_rows(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5},\
                                         {1: '1', 'd': 100, 'finish_time_seconds': 5},\
                                         {2: '2', 'd': 100, 'finish_time_seconds': 10}])
        print('Testing column_velocity with 3 rows...')
        self.assertEqual(column_velocity(dataframe),[300,300,150])
        print("PASS")

    
    def test_with_2_duplicate_columns_same_values(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5,'finish_time_seconds': 5},\
                                         {1: '1', 'd': 100, 'finish_time_seconds': 5, 'finish_time_seconds': 5},\
                                         {2: '2', 'd': 100, 'finish_time_seconds': 10, 'finish_time_seconds': 10}])
        print('Testing column_velocity result with duplicate columns with same values...')
        self.assertEqual(column_velocity(dataframe), [300,300,150])
        print("PASS")
        
    def test_with_2_duplicate_columns_different_values(self):
        dataframe = pd.DataFrame([{0: '0', 'd': 50, 'finish_time_seconds': 5,'finish_time_seconds': 5},\
                                         {1: '1', 'd': 100, 'finish_time_seconds': 5, 'finish_time_seconds': 10},\
                                         {2: '2', 'd': 100, 'finish_time_seconds': 10, 'finish_time_seconds': 10}])
        print('Testing column_velocity result with duplicate columns with different values...')
        self.assertEqual(column_velocity(dataframe), [300,150,150])
        print("PASS")
        
if __name__ == '__main__':
    unittest.main()
